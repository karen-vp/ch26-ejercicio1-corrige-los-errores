Conceptos de Javascript

function
Una función "function" en Javascript es un bloque de codigo diseñado para hacer una tarea en particular.
let
La palabra reservada "let" es una variable que tiene las siguientes caracteristicas: no puede ser redeclarada, debe ser declarada antes de ser usada, tiene un enfoque de bloque.
const
La palabra reservada "const" es una constante, la cual tiene las siguientes caracteristicas: no puede ser redeclarada, no puede ser reasignada, tiene un enfoque de bloque.

Conceptos de HTML

<input>
La etiqueta "input" especifica una caja de texto donde el usuario puede ingresar información.
<a>
La etiqueta "a" define un hipervínculo, que es usado para redirigir a otra página. El atributo más importante de la etiqueta es href, el cual indica el url de destino.
<header>
La etiqueta "header" representa un contenedor para contenido introductorio de la pagina, o el menu de navegación, además puede contener el logo de la empresa.
